# Drakeling Tweaks Features

## Gift of Knowledge stored XP restoration

The mod can restore stored XP gain from tamed and unclaimed creature kills.

Wild creature kill XP remains unchanged.

## Cooldown multipliers

The following Drakeling cooldowns can be configured as percentages:

- Breath cooldown;
- Biome Boost cooldown;
- Gift of Fortune cooldown.

A value of `100` keeps the default cooldown. Lower values reduce the cooldown. Higher values increase it.

## XP info display

The mod can optionally display XP information when stored XP is granted.

## Shoulder vocal sound toggle

While the Drakeling is carried on the shoulder, the radial/menu option can toggle Drakeling vocal sounds on or off.

This is intended to reduce repeated shoulder pet sound spam without replacing the creature.
